# GoGuestBook
A trivial Go-based guest book with moderation

## Features
- [x] Author, mail and text field
- [x] Moderation (approval via email and moderation links)
- [x] Simplistic spam protection (IP-based post rate limit and simple challenge)
- [x] Comment function for the site owner
- [x] Unicode Emoticons

## Non-features
- No admin interface
